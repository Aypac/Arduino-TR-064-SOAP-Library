TR-064 SOAP Library
Library for communicating via TR-064 protocol (e.g. Fritz!Box)

Introduction

This library allows for easy communication of TR-064 (and possibly TR-069) enabled devices, such as Routers, smartplugs, DECT telephones etc. Details, examples and the latest version of this library can be found on the Github page. A descriptor of the protocol can be found here.

Initial version: November 2016
Last updated: Mar 2020

Dependencies

This library depends on:

  • MD5Builder (versions for ESP8266, ESP32)
  • ESP8266HTTPClient (ESP8266) or HTTPClient (ESP32)
